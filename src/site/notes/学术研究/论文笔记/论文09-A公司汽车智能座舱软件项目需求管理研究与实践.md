## 需求管理的分析阶段
{ #900fb0}


按照需求的生命周期，可以分为：
+ 需求的获取与分析
	+ 内部需求获取：需求池子、平台功能路线图
	+ 外部需求获取：自动化VOC系统
	+ 需求分析：QFD+AHP
+ 需求的定义与质量评估（评审）
	+ 评估采用FMEA
+ 需求的跟踪与变更
+ 需求验证和测试
按照这个阶段定义，对于DJI的软件需求管理问题详见[[学术研究/MEM论文选题-需求管理#^42ae34\|MEM论文选题-需求管理#^42ae34]]
