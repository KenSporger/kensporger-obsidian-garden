# 一、小组成员介绍

# 二、项目简介: X公司无人机机库

![无人机库产品展示](/img/user/%E8%81%8C%E4%B8%9A%E6%8E%A2%E7%B4%A2/%E8%B4%A8%E9%87%8F%E7%AE%A1%E7%90%86/assets/%E6%97%A0%E4%BA%BA%E6%9C%BA%E5%BA%93%E8%B4%A8%E9%87%8F%E5%8F%AF%E9%9D%A0%E6%80%A7%E5%88%86%E6%9E%90/%E6%97%A0%E4%BA%BA%E6%9C%BA%E5%BA%93%E4%BA%A7%E5%93%81%E5%B1%95%E7%A4%BA.png)

X公司无人机机库是一款专为无人机打造的无人值守作业基站，集成了自动起降、电能补给、远程调度、数据存储与回传、环境感知与防护等核心功能，可替代人工执行重复性的飞行任务，实现无人机全天候、常态化作业。

该机库具备轻量化、易部署等特点，整机重量34千克，防护等级达IP55级别。集成RTK厘米级定位、4G增强图传、全向避障等技术；搭载温湿度计、风速计、雨量计等传感器。相比一代机库，二代产品采用斜坡归中的V型槽设计提升降落可靠性，同时精简了硬件结构，体积和重量降低了75%和68%。

目前，该机库已广泛应用于应急救援（如火情快速响应）、电力巡检、油气管道监测、城市安防、生态环保等领域。应急场景中可缩短响应时间，为救援争取黄金窗口；巡检场景中可实现偏远区域、复杂地形的常态化监测，减少人工巡检的安全风险与效率损耗；多环境适配能力则让无人机作业覆盖高海拔、沿海盐雾等特殊区域，进一步拓展了无人机的应用边界。

本项目聚焦X公司无人机机库的质量提升与核心结构优化，通过多种质量工程方法开展系统性研究：运用 QFD（质量功能展开）梳理用户需求与技术特性的映射关系，明确降落复位结构、环境适应性等核心技术优先级；借助 TRIZ 理论解决一代机械推杆降落平台寿命短、归位慢的痛点，创新设计 V 字型滑动归中结构；通过 DOE（试验设计）优化 V 型槽的结构形态、阻尼比、表面光洁度等参数，提升不同风速下的降落稳定性；结合 MSA（测量系统分析）、CPK（过程能力指数）与 SPC（统计过程控制）保障核心零件的生产质量；通过单因素方差分析验证降落速度对晃动角度的影响，为飞控参数设定提供数据支撑。最终实现机场降落可靠性、结构耐久性与作业效率的全面提升，满足用户对高稳定性、低故障风险的核心需求。

# 四、TRIZ: V字型滑动归中降落平台设计

## 4.1 问题描述

![伸缩杆降落平台.png](/img/user/%E8%81%8C%E4%B8%9A%E6%8E%A2%E7%B4%A2/%E8%B4%A8%E9%87%8F%E7%AE%A1%E7%90%86/assets/%E6%97%A0%E4%BA%BA%E6%9C%BA%E5%BA%93%E8%B4%A8%E9%87%8F%E5%8F%AF%E9%9D%A0%E6%80%A7%E5%88%86%E6%9E%90/%E4%BC%B8%E7%BC%A9%E6%9D%86%E9%99%8D%E8%90%BD%E5%B9%B3%E5%8F%B0.png)

一代降落平台采用机械推杆实现无人机降落后的精确归中，避免充电时无人机机位不正导致对接出错或舱盖合拢时损伤无人机。但机械推杆方案存在推杆使用**寿命短**、易故障的缺点，且推杆实现无人机归位**耗时长**，无法满足起飞准备时间<=15s的技术特性目标。

## 4.2 问题分析

针对上述问题，采用TRIZ方法进行创新性设计。

需要**改善的技术参数**是机械推杆的耐久和寿命，即参数9——速度（Speed），即物体运动的速率、单位时间的作用率；参数15——移动物体作用时间（Duration of action of moving object），即一个物体作用持续的时间，寿命、失败的平均时间都是对物体持续时间的衡量等。

会**损害的技术参数**是参数12——形状，即系统的外部轮廓或外貌；参数19——运动物体能量利用，即物体做功的容量。在经典的机械体系中，能量是指力和距离的乘积。这里的能量利用具有更广泛的含义，包含做任何事情都需要的能量，甚至还包括了一个超级系统的能量利用。

|   |   |   |
|---|---|---|
|**工程参数**|**12-形状（Shape）**|**19-运动物体能量利用（Use of energy by moving object）**|
|**9-速度（Speed）**|35, 15, 18, 34|8, 15, 35, 38|
|**15-移动物体作用时间（Duration of action of moving object）**|14, 26, 28, 25|28, 6, 35, 18|

查询TRIZ的40个发明原理表，得到以上矛盾矩阵表分别为：

参数9 —— 参数12，对应原理为35（物理或化学参数改变原理）、15（动态特性原理）、18（机械振动原理）、34（抛弃或再生原理）；

参数9 —— 参数19，对应原理为8（重量补偿原理）、15（动态特性原理）、35（物理或化学参数改变原理）、38（强氧化剂原理）；

参数15 —— 参数12，对应原理为14（曲面化原理）、26（复制原理）、28（机械系统替代原理）、25（自服务原理）；

参数15 —— 参数19，对应原理为28（机械系统替代原理）、6（多用性原理）、35（物理或化学参数改变原理）、18（机械振动原理）

## 4.3 解决方案

![V槽滑动归中降落平台.png](/img/user/%E8%81%8C%E4%B8%9A%E6%8E%A2%E7%B4%A2/%E8%B4%A8%E9%87%8F%E7%AE%A1%E7%90%86/assets/%E6%97%A0%E4%BA%BA%E6%9C%BA%E5%BA%93%E8%B4%A8%E9%87%8F%E5%8F%AF%E9%9D%A0%E6%80%A7%E5%88%86%E6%9E%90/V%E6%A7%BD%E6%BB%91%E5%8A%A8%E5%BD%92%E4%B8%AD%E9%99%8D%E8%90%BD%E5%B9%B3%E5%8F%B0.png)

在充分考虑应用场景，以及适用产品的特性后，筛选并运用了适用的原理35（物理或化学参数改变原理），改变物理状态（外形构造）；原理28（机械系统替代原理），结构化场代替非结构场。

二代降落平台采用了V字型滑动归中设计，解决了机械推杆寿命短、归位慢的问题。V字型槽位改变了停机台平面的物理外形，限定了无人机在纵向位置落点，使得横向位置不会偏移。而机库合拢时可以将无人机纵向推至中心，保证无人机更快落到目标位置的同时，也保障了无人机不受损伤。

在V 型槽结构设计时，需要综合考虑诸多因素，使得无人机降落更稳定。核心目标是实现无人机成功降落，在成功降落前提下，最小化最大晃动角度，提升降落稳定性。最大晃动角度定义为无人机接触 V 槽至完全降落过程中，横滚或俯仰的最大角度（通过飞控 IMU 数据获取）。

## 5.2 田口分析参数设定

### 5.2.1 响应特性定义

- 首要响应：成功降落（二分类：是 = 1，否 = 0），属于 “望目特性”（目标 = 1，即 100% 成功）
- 次要响应：最大晃动角度（连续型），属于 “望小特性”（目标 = 0，越小越好）
- 优先级：成功降落＞最大晃动角度，仅对 “成功降落 = 是” 的样本进行晃动角度分析

### 5.2.2 信噪比计算规则

- 成功降落（望目）：SN=10log [(均值 ²)/(方差)]
- 最大晃动角度（望小）：SN=-10log [(Σy²)/n]（y 为晃动角度，n 为重复次数）

## 5.3 Minitab 22.2 操作步骤

### 5.3.1 数据录入与格式设置

1. 打开 Minitab，新建工作表，录入 64 行数据，列定义如下：
    1. C1：试验号（1-64）
    2. C2：结构形态（文本型：I 型、Ⅱ 型、Ⅲ 型、Ⅳ 型）

### 5.4.3 最佳参数组合

|   |   |
|---|---|
|**可控因子**|**最优水平**|

|阻尼比|0.8|

**结构形态：锁定 Ⅲ 型的核心原因：**

- Ⅲ 型是唯一在全风速（2 级、4 级、6 级、8 级）下实现 100% 成功降落的结构形态，满足 “成功降落” 首要目标。
- 从信噪比（SN 值）分析，Ⅲ 型的 SN 值显著高于 Ⅰ 型、Ⅱ 型、Ⅳ 型，表明其在不同风速下的稳定性最强，最大晃动角度波动最小。
- 排除理由：Ⅰ 型、Ⅱ 型在 8 级风下各出现 1 次降落失败，抗极端风载能力不足；Ⅳ 型因侧边大弧度设计，导致无人机无法充分滑动至平台底部，8 级风下出现 1 次降落失败，成功降落率未达 100%。

**阻尼比：选定 0.8 的核心原因：**

- 阻尼比 0.8 的 SN 值最高，对最大晃动角度的抑制效果最优，能有效降低无人机着陆时的冲击，提升降落稳定性。
- 对比其他水平：0.2（低减震）、0.4（中减震）、0.6（高减震）的 SN 值依次递增，但均低于 0.8，减震效果和抗干扰能力不足，无法最大化降低晃动角度。

**表面光洁度：选定 0.4 的核心原因：**

- 表面光洁度 0.4 的 SN 值达到峰值，实现了 “滑动顺畅” 与 “摩擦可控” 的平衡，无滑动过量或卡顿问题，最大晃动角度最小。
- 排除理由：0.2（超高光洁）因摩擦过小导致无人机 “滑动过量”，增加晃动波动，SN 值下降；0.8（常规光洁）、1.6（低光洁）因摩擦过大，导致无人机滑动不顺畅，同样降低稳定性，SN 值均低于 0.4。

# 六、生产质量管理

V型槽的结构形态直接影响了无人机降落的成功率，因此V型槽零件III型结构的控制尤为重要，该V型槽零件采用注塑工艺，其零件的结构质量检测方式为采用精密的3D扫描仪扫描零件得到点云后使用软件拟合，拟合后的轮廓与设计轮廓对齐顶点，再软件计算计算未交叠部分的面积总和，要求总和≤15mm²能够满足要求。

# 七、降落速度对晃动角度的影响分析

在室外压测过程中发现，无人机多次在低风速下发生降落倾翻，这一现象与最初试验设计不符合。分析发现不同任务场景下无人机降落速度差异较大，但尚未明确速度波动对晃动稳定性的量化影响——部分现场数据显示0.9m/s降落时晃动角度偶超20°，0.1m/s降落时存在“滑动不充分”风险，亟需通过单因素分析验证因果关系。

## 7.2 单因素方差分析

1. 打开 Minitab 后，在工作表创建 3 列，分别命名为 “降落速度”“风速等级”“最大晃动角度”。
2. 按试验设计录入 80 组数据，比如降落速度列依次录入 0.1m/s、0.3m/s、0.6m/s、0.9m/s，每个速度在 2/4/6/8 级风速下各重复 5 次，对应填入每次试验的最大晃动角度数据。
3. 主效应分析：构建单因素方差分析模型（因变量：晃动角度；自变量：降落速度），计算F统计量与P值，若P<0.05则表明“降落速度对最大晃动角度存在显著影响”。
4. 多重比较：若主效应显著，采用LSD（最小显著差异法）进行两两比较，明确具体哪两个速度水平间的晃动角度差异显著（如0.6m/s与0.9m/s、0.1m/s与0.3m/s等）。
