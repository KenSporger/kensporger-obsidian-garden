# 知识体系
{ #ca2a69}


## 曾经的尝试

建立一套高效好用、可持续的笔记系统，这是一直以来在不断探索的课题。

大一那会儿，我在博客园写技术文章；后来执着于个性化，又折腾起WordPress、Hexo自建站点，不少时间耗在了网页渲染与美化上。毕业工作后，笔记大多写在了公司电脑和内部Confluence里。直到产生了离职的想法，才转而投向能云同步的Notion，在上面整理读书笔记、沉淀关键的工作思考，期间也动过在知乎、B站总结知识的念头。到了读MEM的阶段，又尝试了通过手帐来做纸质笔记。

一路摸索下来，笔记书写的体验没有提升，几年来的笔记反而散落在各个平台，而曾经认真写下的东西，大多再也没有翻开过。

## 回归记录的本真

今年，笔记系统有了新的改变。我坚持用了一年多flomo，随手记录日常事件和想法，年底接触了Obsidian，体验不错。

其实，flomo的每日回顾、双链功能我并不常用，tag也懒得分级管理。但正如它的宣传语“持续不断记录，意义自然浮现”，flomo带给我的最大价值是思维的改变——让我有感而发时会有记录的冲动（至少每天睡前会想一想，今天有什么值得记的）。至于Obsidian，目前只用了一个月，没去折腾太多复杂功能，只体验了最核心的双链，然后通过插件实现了flomo笔记的导入。

这样一来，flomo记碎片，Obsidian做整合，我似乎回归到了笔记系统的本真。从前执着的博客、知乎、B站，无非是多了一层“展示”的外衣，试图满足自己“被看见”的需求。当然，偶尔这种“展示”也是必要的——并非为了博得关注，而是通过它获得高质量的反馈（新的观点），以此在某个领域不断精进。

这里简单记录一下我对笔记系统的基本需求：
+ 愿意经常去使用；
+ 能够高效串联、整合知识；
+ 系统简单聚焦，平台间能形成联动互补（不强求All In One）；
+ 安全可靠，支持云同步，避免数据丢失；
+ 能覆盖自己的常用场景（MEM笔记、PPT报告、论文分析、代码学习等）；

另外，在大模型兴起的当下，我也考虑过在笔记系统中集成AI的可能性。人受大脑机能的限制，很难全面地回顾曾经的所思所想，深入地了解自己，而AI或许能弥补这一不足——希望它能像一个[[生活随笔/flomo/2025-04-27/memo@2025-04-27\|知心友]]，基于我所有的笔记与我对话，帮我挖掘那些被忽略的思考；又或者更进一步，在我写文章时，能基于我自身的知识储备给予联想与灵感。但正如[@阿冰](https://bbboundary.github.io/2025/01/30/2025_NewPeriod_NewStart/)所说，过度依赖AI会削弱第一大脑的主体地位，导致自身思考能力的下降，所以关于如何合理地利用AI还有待研讨。

总之，笔记系统从来都不是一蹴而就、一劳永逸的，它需要跟着自己的需求慢慢迭代、逐步完善。不必追求一开始就做到完美，也不必耗费大量精力去折腾工具的各类功能，最重要的是先记录起来。[[生活随笔/flomo/2026-02-01/memo@2026-02-01\|memo@2026-02-01]]

## 从领域出发

之前看过[[生活随笔/flomo/2025-02-01/memo@2025-02-01\|少楠关于知识管理的视频]]，他提到几点：
+ 我们应只获取、投资与自身领域相关的内容；
+ 没有领域就没有边界和关注点，容易陷入分不清进展的假忙碌；
+ 即便每年的领域会有变化，也需保持继承性，才能实现知识积累与复利，最终成为领域专家；

过去一年，我在很多个领域投入了时间：游戏设计、创业方法、哲学、心理学、会计、音乐、手帐、农业、股市、宣传制作、工程管理、ROS机器人。回顾这些经历时，我着实被自己的精力分散程度震惊到了。尤其是GAP期间，我每周去旁听几门本科生课程，可一整个学期坚持下来的只有《中西乐理》，其他课程和领域探索基本都是浅尝辄止。直到年底论文和找工作提上日程，我才真正意识到领域聚焦的必要性。

就目前而言，我希望深入探索以下四个领域，除此之外的事情今年会谨慎投入：
+ **器乐**：主要包括长笛练习、古典音乐鉴赏、器乐相关番剧、合奏/演出活动及乐理学习等；
+ **ROS机器人**：作为个人长线项目，以ROS小车为平台，学习自主移动机器人、具身智能相关技术；第二份工作也倾向找这个行业的，希望能将工作中的知识转化到个人项目中；
+ **工程管理**：以完成MEM论文为目标对待工作，结合专业课程所学，在企业中关注需求管理、质量管理、风险管理等方面的问题与解决方案。
+ **哲学/心理学**：主要是针对自己的人格、人生、思维方式的持续探索，以客观地认识自己。

> [!NOTE] AI的建议：个人的精神养分
> 你列了三个聚焦领域（器乐、ROS机器人、工程管理），这个聚焦是对的。但日志里还有大量关于哲学、心理学、人生意义的思考，比如对《爱的艺术》《正义论》的反复咀嚼，对“自我认同”“自卑感”的剖析——这些怎么安放？它们不是你“分心”的证据，反而是你所有技术沉淀的“底座”。它们塑造了你思考问题的方式、你面对焦虑的态度、你理解用户和团队的角度。

## 手帐与电子化的抉择

开学前，亲手设计了一本专属于课堂笔记的手帐本，一学期用下来有些祛魅，感觉唯二的优点就是美观和携带方便。列几个让我比较难受的点：
+ A6slim的版幅实在太小，内页我基本都是裁成B5大小，折成三折横着写（按照周钢老师教的方法，前两折记录老师讲的内容，第三折写自己的想法和思考，后来发现竖着写空间利用率会更高点）；
+ 买的是莫三家的内页，纸是真的好写，但有一定厚度，导致手帐本顶多只能装下三门课的学期笔记；
+ 每次都要取下来写，因为内页塞得满，装回去也很麻烦；
+ 内页用得超级快，每次裁纸费时费力；
+ 没有欲望给手帐做setting，有点浪费了；

当初选择手帐，是冲着那股仪式感去的，希望通过手帐培养对生活的热爱和记录习惯[[生活随笔/flomo/2024-12-23/memo@2024-12-23\|memo@2024-12-23]]。但目前看来，还是那句话——回归记录的本真。所以下学期打算放弃手帐，直接用文件夹来装笔记。

![Pasted image 20260215142340.png|三折内页的手帐笔记](/img/user/%E7%94%9F%E6%B4%BB%E9%9A%8F%E7%AC%94/img/Pasted%20image%2020260215142340.png)

从另一方面来讲，知识管理也要与时俱进，对上世纪的人来说，纸笔是唯一的选择，纯粹、简单。而在智能化、数字化的时代，知识如洪流，几十年前非数字化的管理方法不一定还适用于现在。[[生活随笔/flomo/2025-02-03/memo@2025-02-03\|memo@2025-02-03]]

对我而言，工作上完全电子化没有什么问题。但在上课与科研场景，纸质笔记在即时记录上的那种自如感，仍是键盘无法替代的。拿读论文来讲，我喜欢打印出来阅读：长时间翻阅眼睛不会累，还可以把几页论文在桌上全部铺开，方便比对。但是每篇论文的总结，在我写了六七篇纸质笔记后，还是转向了Obsidian——因为我发现研究同一个主题、反复咀嚼多篇论文时，非常需要双链这样的关联功能，这是纸质永远难以企及的。[[生活随笔/flomo/2025-12-31/memo@2025-12-31#^b70772\|memo@2025-12-31#^b70772]]

所以目前的想法是：即时记录的场合继续用纸质笔记；而那些真正值得沉淀、值得反复咀嚼的知识，再后期整理进电子文档，不让它们静静吃灰。

## 工作经历与经验

步入工作的头两年，自己压根没想过离职的事。因为岗位内容偏业务开发，工作强度高，根本没精力停下来做技术深耕或者工作上的思考总结。等到第三年有了离职的念头，恍然发觉：三年下来，我的能力提升好像没法量化——那些经历，只是经历，没有变成经验。

其实我也记录了很多和代码框架、业务相关的内容，但这些都是低价值、不系统的，只是简简单单的知识搬运。尤其在AI时代，很多操作层面的知识或许已经没有了记录的必要，最有意义的是那些有思想内核、能形成系统的东西。[[生活随笔/flomo/2026-02-04/memo@2026-02-04\|memo@2026-02-04]]

所幸现在发现了这个问题。对于下一份工作，自己要能够保持“[[生活随笔/flomo/2026-02-03/memo@2026-02-03\|随时可Run]]”的状态。争取30岁的时候，能在自己深耕的领域攒下一套像样的积累。

## 原子阅读

去年4月，我彻底抛弃了Kindle，入手了国产的汉王N10mini阅读器，还发了一期[评测视频](https://www.bilibili.com/video/BV1JdTSzCEmN/?spm_id_from=333.1387.homepage.video_card.click&vd_source=1b86e7f99828fa913abe819992eb7fbd)。用到现在，阅读体验至少能达到及格水平了。下半年不工作，少了通勤时间，阅读变得不太规律，不过因为上学一直有其他方面的输入，倒也没什么大碍。

![Pasted image 20260215172235.png|Notion个人图书馆](/img/user/%E7%94%9F%E6%B4%BB%E9%9A%8F%E7%AC%94/img/Pasted%20image%2020260215172235.png)
之前在Notion上整了一个个人图书馆，用来对读过的书做总结，结果完成得不太好。对全书做总结需要花较多时间，容易被各种更重要的事挤掉；就算总结完了放Notion里，时间一久也基本忘得差不多。所以今年打算培养一下“原子化”阅读的习惯。

> [!Note] AI建议：“原子化”阅读
> 你在Notion上做个人图书馆遇到的核心矛盾（总结耗时、被更重要的事挤掉），其实和你之前笔记里记录的“知识存储原子化”理念是相通的。[[生活随笔/flomo/2025-02-03/memo@2025-02-03\|你记过少楠的观点]]：“**卡片化的好处在于限制了篇幅，就像积木一样可以和其他知识混合联系**。”、“**拒绝收藏和搬运，只记录对自己有启发的内容。**”  如果顺着这个思路，或许不必强求对每本书做“完整的总结”。可以把一本书看作一组“知识块”的来源，只摘取当下最有触动、最能与你现有体系链接的几个核心概念或故事，用自己的话重新描述，以卡片形式存入flomo，并打好双链。这样既减轻了心理负担，又让书里的精华真正流入了你的知识网络，而不是变成Notion里的一堆孤立文档。

另外，这学期的《领导力》课程，小组作业就是读书——要求结合书籍和自己的故事写[[职业经验/CSDK3.0团队管理中的领导力哲学\|报告]] ，并在小组内互相[[职业经验/《领导力》课程分享\|讨论]]。这种方式让我不得不在咀嚼书中内容的同时，去反思自身过往的行为，收益比单纯做笔记大得多。[[生活随笔/flomo/2025-12-14/memo@2025-12-14\|memo@2025-12-14]]

> [!NOTE] AI的建议：
> 你提到《领导力》课程小组作业的形式（读书+结合自己故事+小组讨论）让你“主动咀嚼”内容，收益很大。这个观察其实已经触及了“有效输出”的核心：**真正的理解往往发生在“转译”和“交流”中**。  结合你日志里提到的“输出的目的是获取高质量的反馈”，可以进一步想：这种小组讨论的形式，恰好创造了一个安全、聚焦的“反馈场”。在日常阅读中，我们很难复现这种场，但或许可以尝试简化版：每读完一本有触动的书，用三五分钟在flomo里随手录一段语音，或写几句“如果让我向一位朋友转述这本书最触动我的一点，我会说什么？”这比写完整总结轻量得多，却同样能启动“主动咀嚼”的过程。

# 音乐即自由

音乐，毫无疑问是我今年的主旋律，值得好好写一写！

## 序曲：吹响吧！长笛

[[culaccino/致阿冰的第一封信\|欣赏音乐有三种境界]]：第一层是学会聆听，用音乐调节自己的情绪；第二层是演奏，亲手触摸音符，感受创作者的心境或是抒发自己的心意；第三层则是合奏，体验多把乐器共同构筑的和谐。今年，我终于正式迈入了第二重境界。

大四受《京吹》的影响，喜欢上了古典交响乐和管乐器，格外向往那种在社团里一起练习、合奏的氛围。工作的三年里，我听了十多场音乐会，在深圳音乐厅、广州星海音乐厅留下足迹，可因为缺乏必备的音乐素养，对曲子的理解大多流于表面，所以想着余生一定要学一门乐器（其实试过自学电钢琴和吉他，但是因为工作忙都没坚持下来）。

![Pasted image 20260216130851.png|至今去过的音乐会](/img/user/%E7%94%9F%E6%B4%BB%E9%9A%8F%E7%AC%94/img/Pasted%20image%2020260216130851.png)

正好去年6月离职后，时间一下子充裕起来，我便将这个搁置已久的计划提上日程。个人比较喜欢上低音号、圆号、双簧管、长笛、小号的音色，综合考虑之下，我还是选择了便携、相对容易入门的长笛（后来还入了一个百来块的二手UFO）。

![Pasted image 20260216133852.png|blingbling的长笛](/img/user/%E7%94%9F%E6%B4%BB%E9%9A%8F%E7%AC%94/img/Pasted%20image%2020260216133852.png)

受Lily姐演奏的[《Violet Evergarden Suite》](https://www.bilibili.com/video/BV11J41117jj/?spm_id_from=333.337.search-card.all.click&vd_source=1b86e7f99828fa913abe819992eb7fbd)感染，我给这支长笛取名为violet（薇尔莉特）。紫罗兰的花语是“永恒的美与爱”，恰好与长笛悠扬空灵的音色十分契合；同时也算对《紫罗兰永恒花园》的致敬——一次次地练习，如同薇尔莉特那般的“修行”，用这支长笛，去触碰和表达心声，最终与它融为一体，奏出能真正打动人心的乐章。[[生活随笔/flomo/2025-08-23/memo@2025-08-23\|memo@2025-08-23]]

## 练习曲：苦与乐

去年7月，我跟着一位华师大毕业的学姐开始学长笛，到现在快8个月了，每周大概练5天，每天2小时。特别要夸夸交大的琴房，环境超好，午后练琴还能晒到慵懒的阳光，吹累了就听听隔壁练的各种好听的曲子，格外享受。

![Pasted image 20260216144343.png|交大琴房](/img/user/%E7%94%9F%E6%B4%BB%E9%9A%8F%E7%AC%94/img/Pasted%20image%2020260216144343.png)

练琴的心路历程，可以说是乐趣与枯燥并存。

刚入门那段时间，每日的进步肉眼可见，会有很强的新鲜感和成就感，哪怕只是断断续续吹出一段喜欢的旋律，都欢喜地不得了。[[生活随笔/flomo/2025-08-08/memo@2025-08-08\|memo@2025-08-08]]

学到三四个月后，学习的重心慢慢转向音色打磨、手指的配合，进步速度明显慢了下来，也渐渐遇上不少小瓶颈。同一个乐句，有时反复练上几十遍、上百遍，也未必能立刻练好。这段日子，练琴的感受特别依赖当天的状态：状态好时，吹出自己满意的音色，就会格外陶醉；其次，那些不知不觉中突破瓶颈的感觉，也很奇妙——比如六拍子的曲子，之前总是抓不好大拍子，练了一两周，某一天突然就通了，可能就是所谓的“熟能生巧”吧。[[生活随笔/flomo/2025-10-15/memo@2025-10-15\|memo@2025-10-15]] [[生活随笔/flomo/2025-11-25/memo@2025-11-25\|memo@2025-11-25]]

如今，我已经能吹一些篇幅较长的曲子了，像卡门间奏曲、第二圆舞曲，每天练习前吹一遍，畅快至极。年后老师打算带我开始练科勒练习曲，估计又要进入一段漫长的 “黑暗时期” 了。

## 间奏：一个人的和声

长笛是单音乐器，有时候吹多了不免会有些单调，所以在入门之后其实一直想找一个能合奏练习的机会。无奈我太I了，至今没找到情投意合的搭子QWQ，不过还是遇到了一群值得写一写的同好们。

## 变奏：内心的音色

练琴主要还是靠感觉，要放得开自己。心理状态会直接反映在演奏中，所以情感表达和想象力格外重要。比如要想吹好高音，脑海里先要有理想的音色，去想象气息直冲头顶的感觉；乐句的强弱起伏、律动感，也都要靠情感注入才会鲜活。

但情感表达，本就需要丰富的感性体验。越长大，越习惯把情绪藏起来，反而对内心表达多了一层抗拒，这对演奏的表现力是种阻碍。所谓演奏，本就是[[memo@2025-08-09 \|先 “演” 后 “奏”]]，这大概也是小孩子学音乐的天然优势——天真烂漫，又拥有最鲜活的想象力。[[生活随笔/flomo/2025-11-25/memo@2025-11-25\|memo@2025-11-25]]

其实一开始，我担心过自己可能不适合长笛，因为它给人的印象就是活泼外放的个性。但这一个学期下来，我渐渐觉得，学习乐器或许真的能慢慢改变性格。

上课的时候，老师常让我哼唱乐谱、发 “啊” 打开内口腔，吹高音时做出 “惊讶” 的表情；平时在琴房练习，难免会被人听到还很青涩的音色；之前也曾在人前演奏时紧张到嘴抖、音都吹不准；可最后还是在期末轰趴会上，顺利完成了人生第一次独奏。长笛，正在重塑我内心的音色! [[音乐即自由/长笛2个月学习记录\|长笛2个月学习记录]]
